Privacy Policy - Summary

Bandit or its creators are in no way affiliated with Reddit (the service), Reddit Inc or any of its subsidiaries. Bandit accesses user-generated content of Reddit via its API and authenticates the user (if they wish to do so) using OAuth2 authentication. Anonymous usage statistics are collected for improving the quality of the application. The free version of the application serves advertisements. Some of the advertising services may use cookies or collect personal information for behavioral retargeting. Additional information may be gathered by Reddit, Inc. Be advised of their separate privacy policy.

Login Information

  • Bandit authenticates the user (if they wish to do so) to Reddit using Oauth2 authentication. Authentication is not necessary and the services can be accessed anonymously.

  • Bandit never handles users' login credentials, such as usernames or passwords.

  • Bandit acts on behalf of the user after user has granted the rights to do so. What Bandit does on behalf of the user is shown to the user before granting or denying the access.

  • Login credentials such as usernames or passwords are managed by Reddit Inc.

  • Login is done using an integrated WebView, which essentially opens a web page where the user may input their login credentials to Reddit in order to authenticate. This is comparable to accessing Reddit website directly on your browser and as such is covered under Reddit's privacy policy.

  • Bandit holds a renewal token for maintaining authentication information. User can revoke the token by logging out, clearing the application data or removing the application.

  • Settings related to the user account may be synchronized between the application and Reddit.

  • All data transmitted between the application and Reddit servers is done using a secure HTTPS connection.

Information Collection

  • Anonymous usage statistics are collected about the use of the application. This data includes but is not limited to interactions between the user and the user interface, user device make and model, location and possible error cases. Content accessed by the user is not collected.

  • Bandit uses third-party services for information collection. These third parties may access the information that is collected by Bandit.

  • Bandit uses third-party services for advertising. These third parties may access additional information such as cookies and personal information for behavioral retargeting.

  • Bandit does not collect any information that could be used to identify a person.

Data Retention

  • Anonymous usage statistics are stored indefinitely.
  • Data gathered by third-party services may have their own data retention policies.
  • No personal information is gathered and so it is not stored.
  • Information regarding e.g. the authentication is stored only on the user's device. This data can be removed by clearing the application data or by removing the application.

Opt-out

  • Users can opt out from data collection by uninstalling the application.

User Consent

  • By using the services provided by Bandit, You are consenting to our processing of the gathered information.
